Here is the table of contents of this deliverable :

-doc : directory containing various pieces of modelisation dealing with this server.
-lib : directory containing the necessary external jars for this server to function
-logs : directory containing daily logs
-src : directory containing all of this server's source code.

-KappaServer.jar : the runnable jar file for this server.
-KappaServerTest : a console-based runnable jar file used to test the server.

-log4j.properties : the logging properties.
-server.properties : the other properties.

-readme.txt : this file.
